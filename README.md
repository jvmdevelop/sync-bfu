**sync-bfu** is a lightweight async s3 client.

## features

- s3 client
- async
- ui
- tui

## installation

### prerequisites:

- rust 1.70+
- cargo

### from source:

```bash
git clone https://github.com/jvmdevelop/sync-bfu.git
cd sync-bfu/
cargo build # the binary will appear in target/debug/
```

### run from source:

```bash
cargo run
```
